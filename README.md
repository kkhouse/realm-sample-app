# realm-sample-app
## 概要
Realmを使ったメモ帳アプリ
```
* Create: fabボタン押下→ダイアログ入力で新規作成
* Read  : 起動時・データ変更時に読み込み
* Update: Itemの長押し→ダイアログ入力で更新
* Delete: Itemの削除ボタン押下で削除
```
